еӨҡж¬Ўе…ұзәҝжҖ§зҡ„еҪұе“Қ

  • ж–№е·®иҶЁиғҖзі»ж•°пјҲ$variance\ inflation factor$пјү
    • $VIF_i = \frac{1}{1-R_i^2}$
    • еҪ“$VIF>10(or\ 5)$ж—¶пјҢеҸҜд»ҘеҲ еҺ»
    • еҪ“еҸҳйҮҸжҜ”иҫғйҮҚиҰҒпјҢеҲ еҺ»дёҺе®ғзӣёе…ізҡ„еҸҳйҮҸ

з®—жі•йҖүжӢ©

жңҖзЁіеҒҘзҡ„ж–№жі•

  • $SelectFromModel$ з»ҹдёҖжҺҘеҸЈ
  • еҗҺеҸ°з®—жі•
    • $L1-based$
    • $Support\ vector\ based$
    • $Tree\ based$

8.4 е®һи·өпјҡйҮ‘иһҚйЈҺжҺ§

  • ж–№жі•еҹәзЎҖпјҡиҜ„еҲҶеҚЎжЁЎеһӢпјҢйҖ»иҫ‘еӣһеҪ’
  • еҸҳйҮҸеӨ„зҗҶпјҡзҰ»ж•ЈеҢ–
  • еҸҳйҮҸзӯӣйҖүпјҡеҚ•еҸҳйҮҸпјҢеӨҡеҸҳйҮҸ
  • йҮҚиҰҒж–№жі•пјҡ$WOE$дёҺ$IV$
    • $WOE_i = ln(\frac{py_i}{pn_i}) = ln(\frac{y_i/y_T}{n_i/n_T}) = ln(\frac{y_i/n_i}{y_T/n_T})$
    • $IV_i = (py_i-pn_i)*WOE_i$
    • $i = 1$ жҳҜз”·з”ҹ
    • $py_1 пјҡ $з”·з”ҹиҝқзәҰеҚ жҜ”
    • $pn_1 пјҡ $з”·з”ҹдёҚиҝқзәҰеҚ жҜ”
    • $y_1 : $з”·з”ҹиҝқзәҰж•°йҮҸ
    • $n_1 : $з”·з”ҹдёҚиҝқзәҰж•°йҮҸ
    • $y_T : $жҖ»дҪ“иҝқзәҰж•°йҮҸ
    • $n_T : $жҖ»дҪ“дёҚиҝқзәҰж•°йҮҸ
IVиҢғеӣҙ йў„жөӢж•Ҳжһң иӢұж–ҮжҸҸиҝ°
<0.02 еҮ д№ҺжІЎжңү Useless for prediction
0.02~0.1 ејұ Weak predictor
0.1~0.3 дёӯзӯү Medium predictor
0.3~0.5 ејә Strong predictor
>0.5 йҡҫд»ҘзҪ®дҝЎ Suspicious or too good to be true

9 и®ӯз»ғжЎҶжһ¶

9.1 еәҰйҮҸжҢҮж ҮпјҡдәҢеҲҶзұ»й—®йўҳ

дёҖдёӘж ҮеҮҶ

еӣӣдёӘе®ҡд№үпјҲж•°жҚ®-жЁЎеһӢпјү
  • $True-positives$пјҡж•°жҚ®+пјҢжЁЎеһӢ+
  • $False-positives$пјҡж•°жҚ®-пјҢжЁЎеһӢ+
  • $False-negatives$пјҡж•°жҚ®+пјҢжЁЎеһӢ-
  • $True-negatives$пјҡж•°жҚ®-пјҢжЁЎеһӢ-
дёӨдёӘе…¬ејҸ
  • $Precision = True-positives$/ жЁЎеһӢ+
  • $Recall = True-positives$/ ж•°жҚ®+
дёҖдёӘжҢҮж ҮпјҡF1_score

еҸҰдёҖдёӘж ҮеҮҶпјҡROCжӣІзәҝдёҺROCйқўз§ҜпјҲAUROCпјү

дёӨдёӘжҢҮж Ү
  • зңҹжӯЈдҫӢзҺҮпјҲ$TPR$пјүпјҡеҸ¬еӣһзҡ„еҗҢд№үиҜҚ пјҢ$sensitivity$
    • $TPR = \frac{TP}{TP+FN}$
  • еҒҮжӯЈдҫӢзҺҮпјҲ$FPR$пјү$specifity$
    • $FPR = \frac{FP}{FP+TN}$

жү©еұ•пјҡеӨҡеҲҶзұ»й—®йўҳ

  • дёүдёӘжҢҮж ҮпјҡF1_microпјҢF1_macroпјҢF1_weighted

9.2 еәҰйҮҸжҢҮж ҮпјҡеӣһеҪ’

  • $MAE = \frac{1}{N}\sum_{t=1}^N|y_i-\hat{y}|$
  • $MSE = \frac{1}{N}\sum_{t=1}^N(y_i-\hat{y})^2$
  • $RMSE = \sqrt{MSE}$
  • $R^2 = 1-\frac{\sum(y_i-\hat{y})^2}{\sum(y_i-\bar{y})^2} = 1-\frac{MSE(\hat{y},y)}{Var(y)}$
